# Producer Layer

## توضیحات

این پوشه شامل تولیدکنندگان داده (Producers) است که مسئول تولید و ارسال تراکنش‌های مالی به Kafka هستند.

## فایل‌ها

### `darooghe_pulse.py`

Producer اصلی که تراکنش‌های مالی را به صورت real-time تولید می‌کند.

**ویژگی‌ها:**

- تولید تراکنش‌های تصادفی با الگوهای واقعی
- پشتیبانی از انواع مختلف پرداخت (online, pos, mobile, nfc)
- شبیه‌سازی تراکنش‌های موفق و ناموفق
- تولید رویدادهای تاریخی (Historical Events)
- تولید مداوم با نرخ قابل تنظیم

**پارامترهای محیطی:**

- `EVENT_RATE`: نرخ تولید رویداد (پیش‌فرض: 100)
- `PEAK_FACTOR`: ضریب اوج ترافیک (پیش‌فرض: 2.5)
- `FRAUD_RATE`: نرخ تقلب (پیش‌فرض: 0.02)
- `DECLINED_RATE`: نرخ رد تراکنش (پیش‌فرض: 0.05)
- `MERCHANT_COUNT`: تعداد فروشندگان (پیش‌فرض: 50)
- `CUSTOMER_COUNT`: تعداد مشتریان (پیش‌فرض: 1000)

### `darooghe_pulse_original.py`

نسخه اولیه کد از پوشه base_codes - برای مرجع نگهداری شده است.

## نحوه اجرا

```bash
# با تنظیمات پیش‌فرض
python darooghe_pulse.py

# با تنظیمات سفارشی
EVENT_RATE=200 FRAUD_RATE=0.05 python darooghe_pulse.py
```
