\section*{2. Data Engineering and Leakage Control}
The SQL moving-average deliverable is provided in \texttt{code/solutions/q1\_moving\_average.sql}.

Leakage diagnostics show \texttt{Visa\_Approval\_Date} is a direct post-outcome feature:
\begin{itemize}
    \item corr(visa present, migration target) = 1.000
    \item P(Migration=1 $|$ visa present) = 1.000
    \item P(Migration=1 $|$ visa absent) = 0.000
\end{itemize}

Therefore this feature is excluded from training.
